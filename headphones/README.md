# Headphones Webpage

This project is a fully responsive web page built from scratch using only HTML and CSS. No external libraries or JavaScript were used.

## Features

- Fully responsive design (mobile-first approach)
- Accessibility-friendly structure
- Maximum content width: 1000px
- Hover and active states implemented for links and buttons
- Uses custom fonts: Source Sans Pro and Spin-Cycle-OT
